# Pronounciation game
![main](https://github.com/rsmelo92/elocution/actions/workflows/main.yml/badge.svg)

Check your pronounciation skills

## Getting started

Before running it, you must install depedencies with

```
yarn
```

Then just run the project locally with 

```
yarn start
```

## Tests

Just run

```
yarn test
```
